This project scrapes sentences from Yle Selkouutiset, a news site that uses simple Finnish for language learners. Sentences and then words are randomly selected, and then options are generated by lemmatizing the selected word and searching other case forms from Wikisanakirja.

The project uses Python libraries nltk, libvoikko, and BeautifulSoup for tokenizing sentences, lemmatizing Finnish words, and scraping Wiktionary.org for cases. It also uses Flask for web interactivity.
